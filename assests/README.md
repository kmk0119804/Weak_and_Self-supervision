# Assets

Storage for training results, model outputs, and visualization files.

## Purpose

This folder will contain:
- Training logs and curves
- Model checkpoints (organized by training stage)
- Evaluation results and metrics
- Visualization outputs

## Planned Structure

```
assets/

└── self/
    └── runs/
```

## Usage

Training outputs from YOLOv8 `runs/` folder can be automatically moved here for better organization.
